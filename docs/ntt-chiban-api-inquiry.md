| # | 質問カテゴリ | 重要度 | 理由 |

## 備考

- **ちばんAPI が返すデータ**: 住所→地番変換、筆界ポリゴン（zoom 17-21）、地番検索。ポリゴンから敷地の形状と面積が取得可能。
- **代替手段**: 法務省「登記所備付地図」GeoJSON（無料・オープンデータ）もあるが、カバー率約47%、座標精度にばらつきあり。
- **ビジネスモデル案**: ちばんAPIのコストをVolumeCheckの月額サブスクに含め、「住所入力だけで敷地自動取得」を有料プランの付加価値とする。
