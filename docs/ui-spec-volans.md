## 5. 実装ガイドライン

- **禁止**: Lucide**のみ**の使用（原則）。必要なら Phosphor / Heroicons / Tabler を混ぜる
- アイコン: 3Dビューのツールバーは Phosphor（`Cube`, `Square`, `ArrowsOut`, `Ruler`）等
- タブ/ボタン/入力: shadcn素 ではなくカスタムした色+radius
- ゲージ: 半円（180°）SVG path。色は「適合」なら緑、以下なら赤
- 状態管理: page-level useState で OK（要素多いので将来 Zustand 検討）
- デモデータ: 画像の数値を **そのまま** 初期値にする（5,420.18 / 6,712.45 / 1,292.27 / 0.612 / 0.586 等）
- 画像の文言は句読点含め **完全コピー**。例: 「天空率計算は平成14年 国交省告示 第1350号に基づく方法で算定しています。」

## 7. 非機能要件（抜粋）

- モバイル最小幅 360px で破綻しないこと
- デスクトップ最小 1024px で3カラムが保てること
- 3D描画は WebGL 必須、未対応時 2D平面にフォールバック
- オフライン（GitHub Pages 静的版）でも Dashboard / パターン比較 / レポート生成までは動作

## 8. 変更禁止事項

以下はユーザーから「絶対守れ」と明言された正典要素。勝手に変えない:
- プロダクト名/サブタイ: VOLANS / 「最大ボリュームを、一瞬で。」
- 3カードの色分け: 斜線=青 / 天空率=緑 / 増加分=橙
- 斜線envelope=赤点線 / 天空率envelope=緑点線
- 測定点ゲージの数値（デモ: 0.612 / 0.586）
- サマリー数値（5,420.18 / 6,712.45 / +1,292.27 / 8階→10階）
- 最下フッター注記の文言

改変する場合はユーザーに確認して画像を差し替えてもらってから。
