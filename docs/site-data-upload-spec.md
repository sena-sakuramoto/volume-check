# 敷地データ アップロード仕様書

## 概要

VolumeCheckで正確なボリュームチェックを行うために、以下の敷地データが必要です。
お客様には下記のいずれかの方法でデータをご提供いただきます。

---

## 1. 対応フォーマット

### A. 測量図・公図（画像/PDF）— 推奨

| 項目 | 内容 |
|------|------|
| 対応形式 | JPEG, PNG, WebP, HEIC, PDF |
| 最大サイズ | 10MB |
| 推奨解像度 | 300dpi以上 |
| 必要な情報 | 境界点の座標（または距離・角度）、方位、縮尺 |

**AI自動読取対応**: Gemini Vision AIが測量図から境界点座標・面積・道路情報を自動抽出します。

**推奨書類（優先順）:**
1. **確定測量図**（境界確定済み）— 最も正確
2. **現況測量図** — 実測値あり
3. **地積測量図**（法務局備付）— 登記時の測量データ
4. **公図**（法務局備付）— 精度は低いが形状の参考になる

**注意事項:**
- 図面全体が写っていること（切れ・欠けがないこと）
- 文字が判読可能な解像度であること
- 方位記号が含まれていること
- 縮尺の記載があること

---

### B. CADデータ（将来対応予定）

| 項目 | 内容 |
|------|------|
| 対応形式 | DXF, DWG（予定） |
| 座標系 | 平面直角座標系（JGD2011）推奨 |
| レイヤー | 敷地境界線が独立レイヤーであること |

**必要レイヤー:**
- 敷地境界線（閉じたポリライン）
- 道路境界線
- 隣地境界線

---

### C. 座標データ（手動入力/CSV/GeoJSON）

#### CSV形式
```csv
point_no,x,y
1,0.000,0.000
2,12.500,0.000
3,12.500,20.000
4,0.000,20.000
```

| 項目 | 内容 |
|------|------|
| 座標系 | 平面直角座標系（m）または 緯度経度（WGS84） |
| 区切り | カンマ区切り（CSV） |
| 文字コード | UTF-8 |
| 頂点順序 | 時計回りまたは反時計回り（閉合すること） |

#### GeoJSON形式
```json
{
  "type": "Feature",
  "geometry": {
    "type": "Polygon",
    "coordinates": [[[139.700, 35.680], [139.701, 35.680], [139.701, 35.681], [139.700, 35.681], [139.700, 35.680]]]
  },
  "properties": {
    "area": 500.0
  }
}
```

---

### D. 手動入力（アプリ内）

アプリ内の地図上で敷地境界をクリックして多角形を描画できます。
住所検索で対象地付近に移動し、境界点を順にクリックしてください。

## 2. 必須情報

ボリュームチェックに最低限必要な情報：

| 項目 | 必須 | 説明 |
|------|------|------|
| **敷地境界の形状** | ◎ | 多角形の頂点座標（最低3点） |
| **敷地面積** | ○ | 自動計算可能。登記面積があれば照合用に |
| **接道情報** | ◎ | 道路の位置・幅員 |
| **用途地域** | ◎ | 住所から自動判定可能（手動上書きも可） |
| **所在地（住所）** | ○ | 緯度の算出に必要（日影計算の太陽位置に影響） |

### 接道情報の詳細

| 項目 | 必須 | 説明 |
|------|------|------|
| 道路の位置 | ◎ | 敷地のどの辺に接しているか |
| 道路幅員 | ◎ | メートル単位（例: 6.0m） |
| 道路の種類 | △ | 建築基準法上の道路種別（42条1項等） |
| 道路中心線位置 | △ | セットバックが必要な場合 |

---

## 3. 任意情報（精度向上に有効）

| 項目 | 説明 |
|------|------|
| 地盤高さ（GL） | 敷地の高低差がある場合 |
| 高低差 | 道路と敷地の高低差 |
| 隣地の状況 | 隣地側の建物高さ等 |
| 指定建蔽率・容積率 | 用途地域から自動取得されるが、特定行政庁の指定値が異なる場合 |
| 日影規制の詳細 | 測定面の高さ（1.5m/4.0m）、規制時間 |
| 防火地域指定 | 防火地域・準防火地域 |
| 高度地区指定 | 第一種〜第三種高度地区 |
| 地区計画 | 地区計画による制限がある場合 |

---

## 4. データ品質の目安

| レベル | データソース | 精度 | 推奨用途 |
|--------|-------------|------|----------|
| ★★★ | 確定測量図 | ±数cm | 最終ボリュームチェック |
| ★★☆ | 現況測量図・地積測量図 | ±10cm程度 | 設計初期検討 |
| ★☆☆ | 公図・ブルーマップ | ±1m程度 | 概算レベル |
| ☆☆☆ | 地図上手動入力 | ±数m | ざっくり検討 |

> **注意**: VolumeCheckの計算結果は参考値です。確認申請等の法的手続きには、必ず確定測量図に基づく正式な検討を行ってください。
